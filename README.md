# mvtui

TUI for Mullvad WireGuard on Linux.

## Install

```
sudo make install
```

Then just run `mvtui`.

## Uninstall

```
sudo make uninstall
```

## Keys

- j/k - navigate
- Enter - connect
- d - disconnect
- a - toggle autostart
- i - setup (add device)
- r - refresh servers
- q - quit

## Requirements

- wireguard-tools
- Mullvad account
